# 第一部分：发文原文要求（人行规范层）

> **用于回答"这个表要求报送什么"、"新增业务应该报到哪个表"等问题**

## 1. 官方报送范围

本表报送报告期内科技贷款的发生额信息，按科技贷款口径逐笔统计报告期内的贷款发放金额。

## 2. 纳入与排除口径

| 方向 | 内容 |
|------|------|
| ✅ 纳入 | 报告期内发生（发放）的符合科技标识的贷款业务 |
| ❌ 排除 | 非科技标识贷款、存量贷款 |

## 3. 关键业务要素

报送维度：客户、机构、科技分类、发生金额、利率等。

## 4. 业务筛选条件

### 4.1 科技企业识别

通过`SMTMODS.PUB_KJDK`科技名单匹配。

### 4.2 发生额时间过滤

按报告期月份筛选贷款放款日期（DRAWDOWN_DT）。
