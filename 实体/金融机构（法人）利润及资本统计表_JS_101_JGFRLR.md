# 第一部分：发文原文要求（人行规范层）

> **用于回答"这个表要求报送什么"、"新增业务应该报到哪个表"等问题**

## 1. 官方报送范围

本表统计金融机构（法人层面）的利润及资本相关指标，包括营业收入、营业支出、利润总额、净利润、资本充足率、一级资本净额、核心一级资本净额等关键指标。

## 2. 纳入与排除口径

| 方向 | 内容 |
|------|------|
| ✅ 纳入 | 法人层面的利润表项目和资本充足率相关指标。 |
| ❌ 排除 | 分支机构单独报送的指标。 |

## 3. 关键业务要素

利润类指标（营业收入、营业支出、利润总额、所得税、净利润等）；资本类指标（资本净额、风险加权资产、资本充足率等）。
