# 第一部分：发文原文要求（人行规范层）

> **用于回答"这个表要求报送什么"、"新增业务应该报到哪个表"等问题"

## 1. 官方报送范围

数字经济产业贷款是指金融机构向数字经济核心产业相关的企业或项目发放的贷款。本表报送报告期末存续的数字经济产业贷款存量数据，与G0107核对。

## 2. 纳入与排除口径

| 方向 | 内容 |
|------|------|
| ✅ 纳入 | 符合数字经济产业标识的存量贷款（通过行业分类或企业标识识别） |
| ❌ 排除 | 非数字经济产业标识贷款 |

## 3. 关键业务要素

报送维度：客户、行业分类、数字产业分类、余额及利率等。

## 4. 业务筛选条件

### 4.1 数字经济产业识别

通过`L_CUST_C`（对公客户补充信息表）中的客户行业代码（indust_stg_type等字段）识别数字经济核心产业。

### 4.2 存量数据过滤

```sql
WHERE T.LOAN_ACCT_BAL > 0
  AND T.ACCT_STS <> '3'
  AND T.CANCEL_FLG <> 'Y'
  AND T.LOAN_STOCKEN_DATE IS NULL
```
