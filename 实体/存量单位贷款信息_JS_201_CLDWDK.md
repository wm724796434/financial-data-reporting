# 第一部分：发文原文要求（人行规范层）

> **用于回答"这个表要求报送什么"、"新增业务应该报到哪个表"等问题**

## 1. 官方报送范围

存量单位贷款指金融机构报告期末存续的、在资产负债表内核算的**非同业单位贷款**，**不含**票据融资及已核销、剥离、转出的贷款。

## 2. 纳入与排除口径

| 方向 | 内容 |
|------|------|
| ✅ 纳入 | 按官方报送范围采集规范定义的相关业务 |
| ❌ 排除 | 票据融资、已核销贷款、已剥离贷款、已转出贷款 |

## 3. 关键业务要素（要求报送的内容维度）

报送机构及经办机构识别；交易主体/客户主体识别；合同、协议或资产唯一标识；金额、币种及折人民币计量；利率与计息要素；起止日期、到期日或实际终止日；地区、行业、部门及企业属性分类。

> 完整原文详见：[单位贷款基础数据_报送指引.md](../参考资料/报送指引/单位贷款基础数据_报送指引.md)

## 1. 重点字段填报说明

以下针对存量单位贷款信息报表的28个字段，提炼报送指引中的核心填报规则和注意事项。

### 1.1 机构信息类

| 字段 | 核心填报要点 |
|------|------------|
| **金融机构代码** | 填报经办机构统一社会信用代码。总行营业部/总行部门与总行共用同一代码的，填总行代码；无分支机构则直接填法人代码 |
| **内部机构号** | 必填字段，不可空置。无分支机构的财务公司、金融租赁公司等填法人的金融机构代码 |
| **金融机构地区代码** | 填报经办机构注册地区划代码（GB/T 2260 6位县区级数字码）。如发现地区代码无法通过系统校验，可能是业务系统未及时更新行政区划信息，确属正确报送的建议反馈至人行审核人员 |

### 1.2 客户信息类

| 字段 | 核心填报要点 |
|------|------------|
| **借款人证件类型/代码** | 境内机构优先填报统一社会信用代码（18位）；少数未三证合一的可填组织机构代码（9位，不含"-"）；境外机构填LEI码（全球法人识别编码），暂无LEI码的可填Z99-自定义码+机构自行设定的唯一编码 |
| **借款人国民经济部门** | 必须填报最底层类别。注意：个体工商户和小微企业主属于住户部门，应在个人贷款中报送，不在单位贷款中报送；融资租赁公司、典当行等地方金融组织属于非金融企业部门，非金融机构部门 |
| **借款人行业** | 采用GB/T 4754最新标准的大类。境外借款人填"200-境外"；"100-个人"不适用于单位贷款 |
| **借款人地区代码** | 境内客户填注册地6位县区级数字码（GB/T 2260）。境外客户前3位填"000"，后3位填GB/T 2659国别代码。港澳台客户按境外处理，不得填810000/820000等行政区划代码 |
| **借款人经济成分** | 填值域树形结构的叶节点（最小类别）。实在不能确定的，允许回湖一级。境外非居民不需填报。广义政府机关/事业单位/社会团体按经费来源和管理方式划分 |
| **借款人企业规模** | 非企业类单位（机关、事业单位、社会团体等）填CS05-其他，不得归入大中小微类型。民办非企业法人、农民专业合作组织应做划型（CS01-CS04），不能简单填CS05 |

### 1.3 贷款信息类

| 字段 | 核心填报要点 |
|------|------------|
| **贷款借据编码** | 必须是唯一编码。不唯一时有两种处理方式：①填本机构唯一识别码；②在相同编码后加"_1""_2"等后缀。需保证在各期报文中保持不变，且与发生额信息中的编码一致 |
| **贷款产品类别** | 贷款是最常见的金融工具。再贷款、黄金借贷、打包信贷受让资产及其他贷款不在报送范围。经营贷款、固定资产贷款、透支、贸易融资、融资租赁、并购贷款、垫款属于报送范围。消费贷款、拆借、回购/返售一般不应在单位贷款中出现 |
| **贷款实际投向** | 采用GB/T 4754最新标准的中类。境外借款主体填"2000-境外"，"1000-个人"不适用于单位贷款 |
| **贷款发放日期** | 填报借据中约定的发放日期，与授信审批时间无关 |
| **贷款到期日期** | 填报借据中约定的到期日期。若分期还款，填最后一次还款日期。**注意：展期、缩期不改变贷款到期日期的报送**（到期日期不应变动，展期到期日期另有专门字段） |
| **贷款展期到期日期** | 展期贷款必填，且应大于贷款到期日期；缩期贷款也需填此字段，且应小于贷款到期日期 |
| **贷款余额** | 填报账面余额（含利息调整、公允价值变动等会计调整），仅报送本金不包含利息。存量贷款余额应>0。因计提减值导致账面余额降为0的，应报减值前的实际本金余额 |
| **贷款余额折人民币** | 必填。外币按会计核算汇率折算；人民币直接填余额 |
| **利率是否固定** | 根据贷款合同中金融机构与借款人的约定填报。存续期内只要利率**有可能**发生变化（挂钩某一基准利率等），无论实际是否变动，均填RF02-浮动利率 |
| **利率水平** | 填报报告日贷款实际执行的年利率水平（百分数，不带"%"号，保留5位小数，如5%→5.00000）。一般应>1%且<相应期限LPR的4倍。**注意：停息贷款的利率水平不应填0**，应按原始借据约定的利率填报 |
| **贷款定价基准类型** | 固定利率和浮动利率贷款均需填报。绝大部分贷款已转换成LPR定价，若存在大量以贷款基准利率为定价基准的贷款，应重点核查业务真实性 |
| **贷款利率重新定价日** | 浮动利率贷款：填最近可重新定价的日期；如下一个可重新定价日晚于贷款到期日，按到期日填。固定利率贷款：填原始到期日期（展期后填展期到期日） |
| **贷款财政扶持方式** | 获财政扶持的贷款必填最底层类别。以再贷款/再贴现资金发放的贷款属于货币政策支持，不属于财政扶持，此字段应空置。一笔贷款同时享受多级贴息补的，填C-联合贴息 |

### 1.4 担保与质量类

| 字段 | 核心填报要点 |
|------|------------|
| **贷款担保方式** | 需填至最底层类别。一笔贷款同时含两种及以上担保方式（抵押、质押、保证）的，填报"E-组合担保"。含房地产抵押的组合担保统一填"E01-其中：含房地产抵押的组合担保"。既有信用/免担保又仅有单一担保方式的，填该担保方式，不得填组合担保 |
| **是否首次贷款** | 存量单位贷款信息中**此字段空置**，且无须作出说明 |
| **贷款质量** | 五级标准分类，与报送金融监管部门口径保持一致 |
| **贷款状态** | 先逾期后展期→按展期报；先展期在展期内逾期→按逾期报；逾期后正常还款→恢复展期。延期还本付息（签订延期协议未签展期合同）→按延期后到期日填、状态按正常 |
| **逾期类型** | 仅有贷款状态为"逾期"时才需填报。分期付息一次性还本的，利息逾期填利息逾期；分期还款的按实际逾期情况报（本金及利息均可能逾期） |
| **贷款用途** | 按合同中约定的用途填报，3000字节以内，不应出现"?""#"""等特殊符号 |
| **登记交易场所代码** | 仅贷款产品类别为其他债权时报送。不涉及公开市场交易的可空置，并填写提交说明 |

## 2. 常见问题及应对

以下提炼报送指引第二节"常见问题及解答"中与存量单位贷款信息相关的核心问题：

### 2.1 客户信息类问题

| 问题 | 解答要点 |
|------|---------|
| **借款人证件代码含"-"怎么办？** | "-"属于特殊字符，统一社会信用代码和组织机构代码不应含"-" |
| **内部机构号对无分支机构的法人怎么填？** | 必填字段不可空置，无法人的金融机构代码即可 |
| **事业单位的国民经济部门怎么填？** | 事业单位→机关团体 |
| **央行企业、省属企业的地区代码怎么填？** | 不可填省一级代码，必须按注册地址填具体的县（区）级数字码 |
| **"偿还借款""流动资金"等用途的贷款投向怎么填？** | 可参考贷款企业主体所属行业填报 |
| **广义政府借款人企业规模怎么填？** | 应填CS05-其他，不得归入大中小微类型。民办非企业法人应做划型（CS01-CS04） |
| **境外分行开展的贷款业务是否纳入？** | 不纳入。境内金融机构境外分行视同境外金融机构，其贷款业务不作为境内金融机构单位贷款报送 |

### 2.2 贷款信息类问题

| 问题 | 解答要点 |
|------|---------|
| **贷款借据编码为空，能否填合同编码？** | 未签订借据的应填报贷款提款书等载明贷款发放指令的凭证编码 |
| **展期合同有多个借据，展期到期日期怎么填？** | 以单笔借据为单位逐笔填报，按各个借据约定的展期到期日期分别填报 |
| **贷款到期未展期，只签了延期协议，状态怎么报？** | 根据延期协议约定的到期日填报贷款到期日期，贷款状态为正常 |
| **停息贷款的利率水平是否填0？** | **不应填0。**贷款停息只是暂停按期计息、收息的账务处理，未重新签订合同，应按原始借据中约定的利率填报 |
| **有贴息的贷款利率填贴息前还是贴息后？** | 填贴息后利率，即根据借据约定向客户实际收取执行的年利率水平 |
| **与合作机构联合贷款的利率怎么填？** | 填报对客利率，即根据借据约定向客户实际收取执行的年利率水平，而非银行实际分得的协议利率 |
| **逾期贷款的利率是否包括罚息利率？** | 应包括罚息利率。对于部分逾期的贷款，应按加权利率填报 |
| **无还本续贷是否属于置换旧债？** | 按实质重于形式原则判断。对于贷款到期后出现特定问题不能按时收回又重新发放的，无论形式如何（借新还旧、续贷、倒贷等），均属于置换旧债。无还款问题的正常经营企业接续授信，不属于置换旧债 |

### 2.3 特殊业务类问题

| 问题 | 解答要点 |
|------|---------|
| **贸易融资（福费廷）的贷款余额按什么填？** | 按账面余额填报。因利息调整等因素造成的账面价值变动不需要报送发生额信息 |
| **其他债权业务的贷款余额按合同金额还是账面金额？** | 按账面价值填报 |
| **法人透支账户怎么区分报送？** | 使用透支额度时透支部分报单位贷款发生额。月末结余为正数报存量单位存款，为负数报存量单位贷款 |
| **境内机构在银登中心登记的资产转出是否属于资产证券化？** | 应按非标准化债权类资产在单位贷款信息中报送，填报实际融资人客户信息 |

## 3. 典型案例

以下提炼报送指引第三节"典型案例分析"中与存量单位贷款信息相关的核心案例：

### 案例一：借款人错报为住户部门

**错误：** 单位贷款中将借款人国民经济部门填为住户部门（D02-为住户服务的非营利机构），或将个体工商户贷款报入单位贷款。

**要求：** 个体工商户和小微企业主属于住户部门，其贷款应在个人贷款中报送。

**处理：** 核查表中住户部门有贷款余额的→修正国民经济部门或调整统计口径。

### 案例二：借款人错报为金融机构部门

**错误：** 将小额贷款公司、融资租赁公司、地方资产管理公司等地方金融组织的国民经济部门错划分为金融机构部门；或二级福费廷业务未穿透报送实际融资企业。

**要求：** 未持有金融牌照的地方金融组织属于非金融企业部门。二级福费廷需穿透报送，借款人仍为最初的融资客户。

**处理：** 地方金融组织→修正为非金融企业部门；二级福费廷→穿透报送出口商信息。

### 案例三：企业规模划分错误

**错误：** 将广义政府借款人（机关、事业单位等）企业规模归入大中小微类型；或将民办非企业法人的企业规模划分为其他（CS05）。

**要求：** 非企业类单位填CS05-其他；民办非企业法人应按划型标准划分CS01-CS04。

### 案例四：产品类别错报为融资租赁

**错误：** 商业银行将融资租赁保理业务的产品类别错报为融资租赁。

**要求：** 商业银行不能直接办理融资租赁业务。融资租赁保理业务的实质是保理，产品类别应为贸易融资。

### 案例五：停息贷款利率错报为0

**错误：** 违约逾期停息处理的贷款利率水平填报为0。

**要求：** 停息只是暂停按期计息的账务处理，未重新签订合同，应按原始借据中约定的利率填报。

### 案例六：利率水平极值异常

**错误：** 未采用百分数报送（如5%报为0.05），或业务系统录入错误导致利率异常。

**要求：** 百分数表示，保留5位小数，一般应>1%且<LPR的4倍。

### 案例七：未统计报送组合担保

**错误：** 对于两种及以上担保方式的贷款按主担保方式填报，未统计组合担保。

**要求：** 涉及两种及以上担保方式的填E-组合担保。含房地产抵押的组合担保填E01。

### 案例八：错报贷款到期日期

**错误：** 根据展期、收回情况调整贷款到期日期，造成跨期校验差额。

**要求：** 贷款到期日期应填报借据中约定的到期日期，展期、缩期不改变此字段报送。展期到期日期另有专门字段。

### 案例九：错报非居民借款人证件类型

**错误：** 将非居民借款人证件类型统一归为A03-其他。

**要求：** 境外机构优先填LEI码。暂无LEI码的填Z99-自定义码。A03-其他仅适用于由境内外其他主管机构赋予的身份证明文件类型。
