# 第一部分：发文原文要求（人行规范层）

> **用于回答"这个表要求报送什么"、"新增业务应该报到哪个表"等问题**

## 1. 官方报送范围

存量单位贷款指金融机构报告期末存续的、在资产负债表内核算的**非同业单位贷款**，**不含**票据融资及已核销、剥离、转出的贷款。

## 2. 纳入与排除口径

| 方向 | 内容 |
|------|------|
| ✅ 纳入 | 按官方报送范围采集规范定义的相关业务 |
| ❌ 排除 | 票据融资、已核销贷款、已剥离贷款、已转出贷款 |

## 3. 关键业务要素（要求报送的内容维度）

报送机构及经办机构识别；交易主体/客户主体识别；合同、协议或资产唯一标识；金额、币种及折人民币计量；利率与计息要素；起止日期、到期日或实际终止日；地区、行业、部门及企业属性分类。

## 3. 业务筛选条件（代码实际取数范围）

### 3.1 会计科目范围（核心过滤条件）

```sql
SUBSTR(T.ITEM_CD, 1, 4) IN ('1303', '1305', '1306')
```

| 科目代码 | 说明 |
|----------|------|
| 1303 | — |
| 1305 | — |
| 1306 | 垫款科目（20240926新增） |

### 3.2 客户类型范围

```sql
INNER JOIN L_CUST_C_TMP G ON T.CUST_ID = G.CUST_ID
  AND G.CUST_TYP <> '3'  -- 去除个体工商户
```

| 客户类型 | 是否纳入 |
|----------|---------|
| 对公单位客户（非个体工商户） | ✅ 纳入 |
| 个体工商户（CUST_TYP='3'） | ❌ 排除 |

### 3.3 排除条件

| 条件 | 说明 |
|------|------|
| `T.LOAN_ACCT_BAL > 0` | 余额大于0 |
| `T.CANCEL_FLG = 'N'` | 剔除核销数据 |
| `T.LOAN_STOCKEN_DATE IS NULL` | 资产未转让（20250311新增，需求JLBA202408200012） |

### 3.4 产品类型映射（代码级）

代码通过 `ACCT_TYP` 字段将贷款借据映射为发文要求的产品类别码（F码）：

|-----------|---------|-------------------|
| F052 | 担保垫款 | ACCT_TYP = '0901' |
| F041 | 账户透支 | ACCT_TYP = '0801' |
| F09 | 融资租赁 | ACCT_TYP = '05' |
| F022 | 经营贷款 | ACCT_TYP = '0202' 或 LIKE '0102%' 或 (070101 + 转贷A) |
| F023 | 固定资产贷款 | ACCT_TYP LIKE '0201%' 或 (070101 + 转贷B) |

| F082 | 国内贸易融资 | ACCT_TYP LIKE '0402%' 或 (070101+转贷E) 或 (1305+人民币) |
