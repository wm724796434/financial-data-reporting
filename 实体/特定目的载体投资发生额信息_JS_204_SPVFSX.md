# 第一部分：发文原文要求（人行规范层）

> **用于回答"这个表要求报送什么"、"新增业务应该报到哪个表"等问题**

## 1. 官方报送范围

特定目的载体（SPV）投资是指金融机构投资的特定目的载体产品，如理财产品、信托计划、资产管理计划等。本表报送报告期内的 SPV 投资发生额信息。

## 2. 纳入与排除口径

| 方向 | 内容 |
|------|------|
| ✅ 纳入 | 按官方报送范围定义的相关 SPV 投资业务。 |
| ❌ 排除 | 未在官方口径中明确排除的，应按采集任务和字段规则判断是否纳入。 |

## 3. 关键业务要素

报送机构识别；SPV 产品信息；投资金额、币种及折人民币计量；起止日期等。
