# 第一部分：发文原文要求（人行规范层）

> **用于回答"这个表要求报送什么"、"新增业务应该报到哪个表"等问题"

## 1. 官方报送范围

本表为科技贷款的汇总统计口径，不含票据直贴。统计科技贷款（不含票据直贴）的汇总金额和加权平均利率，作为五篇大文章的汇总指标。

## 2. 纳入与排除口径

| 方向 | 内容 |
|------|------|
| ✅ 纳入 | 存量科技贷款信息(3.1字段TNA01)中不包含票据直贴的数据 |
| ❌ 排除 | 票据直贴部分（T13字段） |

## 3. 关键业务要素

报送维度：存量科技贷款（纯贷款口径）汇总余额、加权平均利率、贷款笔数。

## 4. 字段清单

完整字段清单详见原文文件。

# 第二部分：代码取数业务范围（实现层）

> **用于回答"这个表怎么取数"、"取了哪些业务"、"业务变更对金数有什么影响"等问题"

## 1. 数据生成程序

该报表无独立的加工层存储过程，由存量科技贷款(3.1)经应用层特殊处理生成，与4.1共用同一SQL，仅不含票据直贴部分。

## 2. 数据基础

- 基础数据源：`PBOCD_JS_201_HDACLKJDK`（存量科技贷款表）
- 排除范围：票据直贴数据（field_type='T13'以外的数据）

## 3. 应用层特殊处理

与4.1（含票据直贴）共用`五大篇章-18科技贷款刷汇总金额利率.sql`，通过筛选 field_type 维度区分含/不含票据直贴的口径。

## 4. 输出指标

- `BALANCE_SUM` — 汇总金额（不含票据直贴）
- `INT_RATE_WA` — 加权平均利率
- `GET_LOAN_NUM` — 贷款笔数
