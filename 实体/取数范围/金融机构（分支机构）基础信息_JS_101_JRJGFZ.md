# 金融机构（分支机构）基础信息_JS_101_JRJGFZ — 取数范围

> 来源程序: `bsp_sp_js_101_jrjgfz.prc`

## 数据源表（FROM）

- PBOCD_JS_101_JRJGFZ_SQ AS SQ1
- SMTMODS.L_PUBL_ORG_BRA AS B

## 关联条件（JOIN）

- INNER JOIN SMTMODS.L_PUBL_ORG_BRA AS B ON SUBSTRING(A.UP_ORG_NUM, 1, 2) || '0000' = B.ORG_NUM AND B.DATA_DATE = IS_DATE
- INNER JOIN PBOCD_JS_101_JRJGFZ_SQ AS SQ1 ON A.ORG_NUM = SQ1.ORG_NUM AND SQ1.CJRQ = VS_LAST_TEXT

## 筛选条件（WHERE）

1. `A.DATA_DATE = IS_DATE`
2. `A.BUSI_STATE = '01'`
3. `EXISTS(SELECT * FROM SYS_OFFICE AS C WHERE A.ORG_NUM = C.ID)`
4. `SUBSTRING(A.ORG_NUM, 5, 2) <> '00' OR SUBSTRING(A.ORG_NUM, 3, 4) = '0000'`
5. `NOT A.ORG_NUM IN ('000000', '510000', '520000', '530000', '540000', '550000', '560000', '570000', '580000', '590000', '600000', '990000', '019801', '029801', '039801', '049801', '059801', '099801', '109801', '119801', '029804', '012102', '012103', '012104', '012105', '012106', '012107', '012108', '120000', '120101', '012502')`
