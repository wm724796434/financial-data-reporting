## 1. 官方报送范围

单位贷款发生额指金融机构报告期内发放、收回、发放又收回的非同业单位贷款（不含票据融资），包括报告期内核销、剥离、转让的贷款。贷款展期不视为新发放。以贷款合同下单笔借据为单位逐笔报送。

## 2. 纳入与排除口径

| 方向 | 内容 |
|------|------|
| ✅ 纳入 | 包括报告期内核销、剥离、转让的贷款 |
| ❌ 排除 | 不含票据融资 |

## 3. 关键业务要素（要求报送的内容维度）

报送机构及经办机构识别；交易主体/客户主体识别；合同、协议或资产唯一标识；金额、币种及折人民币计量；利率与计息要素；起止日期、到期日或实际终止日；地区、行业、部门及企业属性分类。

# 第三部分：报送指引（人行规范层——填报操作指南）

> **用于回答"这个字段具体怎么填"、"哪些情况容易报错"、"常见问题如何处理"等问题**
>
> 来源：《金融基础数据统计报送指引与典型案例汇编》第二章单位贷款基础数据
> 完整原文详见：[单位贷款基础数据_报送指引.md](../参考资料/报送指引/单位贷款基础数据_报送指引.md)

## 1. 报送范围及总体要求

单位贷款发生额信息与存量单位贷款信息共用大部分字段填报规则（详见存量单位贷款信息实体文件的第三部分），以下仅列示发生额信息的**特有规则**：

| 要点 | 说明 |
|------|------|
| **报送范围** | 报告期内发放、收回、发放又收回的非同业单位贷款（不含票据融资），包括报告期内核销、剥离、转让的贷款 |
| **报送粒度** | 以贷款合同下单笔借据为单位逐笔报送。同一借据有多笔放款或收款的，通过交易流水号判断每笔交易的唯一性，逐笔交易报送 |
| **收息不报送** | 仅收回利息的贷款不需要报送 |
| **展期不视为新发放** | 贷款延期、展期不视为新发放，无需报送发生额 |
| **结清处理** | 结清贷款余额为0，存量中不报送，但发生额中应报送收回方向交易信息 |
| **贷款发放日期** | 填报借据中约定的贷款发放日期，**不是**授信起始日期，与授信审批时间无关 |
| **贷款到期日期** | 填报借据中约定的到期日期。**注意：发生额中的到期日期应与存量中的保持一致**，不因提前还款或展期而改变 |
| **贷款实际终止日期** | 对于提前结清、核销、剥离、转让的贷款，需要在发生额中填报此字段，填报贷款实际终止日期 |

## 2. 重点字段补充说明

发生额信息与存量单位贷款信息的字段大部分相同，以下仅列补充说明：

| 字段 | 补充要点 |
|------|---------|
| **贷款产品类别** | 与存量口径一致。其他债权业务的贷款发放日期可按起息日填报，到期日期可按兑付日填报，借据编码/合同编码按唯一识别编号填报，利率水平按票面利率或预期年化收益率填报 |
| **贷款财政扶持方式** | 贷款发生额中的收回信息也需要进行财政扶持方式的判断，与实际情况保持一致——何时发生财政扶持，则何时填报，发放、收回均需判断 |
| **是否首次贷款** | 判断标准：仅看借款人是否首次与**本报送机构**发生信贷关系，不看他是否在其他金融机构有贷款 |
| **银团贷款** | 按金融机构实际参与的贷款发放人金额和余额填报。授信额度和已用额度仅统计金融机构自身对客户的授信数据 |
| **债权融资计划** | 贷款发放日期按起息日报，贷款到期日期按兑付日报，展期到期日按实际展期日报，利率水平按票面利率或预期年化收益率报 |

## 3. 常见问题及应对

以下提炼与单位贷款发生额信息直接相关的常见问题：

| 问题 | 解答要点 |
|------|---------|
| **发生额中的贷款发放/到期日期是授信起止日期吗？** | **不是。**与授信审批时间无关，应填报贷款借据中约定的贷款发放日期和到期日期 |
| **按季/按月付息到期还本，定期付息是否需要报发生额？** | **不需要。**仅当贷款本金发生变动时需要报送发生额信息 |
| **当月发放的业务当月末入账，可以归在当月吗？** | 应按业务实际发生时点报送。当月正常发放的贷款，发放日期应在报告期当月，不应将上月末发放的业务延后至次月 |
| **无还本续贷是否需要报在置换旧债中？** | 按实质重于形式原则。到期后出现特定问题不能按时收回又重新发放的贷款（借新还旧、续贷等），属于置换旧债报送范围 |
| **置换民间借贷是否属于置换旧债？** | **不属于。**被置换旧债范围为金融债务，不包括股东/关联方借款、企业间应收款项、保理公司融资资金等 |
| **新发放贷款置换旧债时，被置换债务金额填多少？** | 填报新发放贷款时点被置换债务实际总金额（原债务剩余金额），如有未偿还的罚息、利息应包含在内 |

## 4. 典型案例

以下案例报送指引中同时涉及存量单位贷款信息和单位贷款发生额信息的相关案例：

### 案例八：错报贷款到期日期

**涉及报文：** 存量单位贷款信息、**单位贷款发生额信息**

**错误：** 根据贷款展期、收回情况，将贷款到期日期错误调整为贷款展期到期日期或贷款实际终止日期。如：
- 经营贷款展期12个月，误将到期日期和展期到期日期均报为展期后日期，导致跨期计算为24个月
- 固定资产贷款提前还款结清，误将到期日期按提前还款日期报送，导致收回方向的贷款期限计算错误

**要求：** 贷款到期日期应填报贷款借据中约定的到期日期。一笔贷款存续期间，除签订延期协议等特殊情况外，到期日期一般不应有变化，**展期、缩期均不改变贷款到期日期的报送**。

**处理：** 按展期约定填报展期到期日期（有专门字段），按实际终止日期填报贷款实际终止日期（有专门字段），到期日期不应变动。
