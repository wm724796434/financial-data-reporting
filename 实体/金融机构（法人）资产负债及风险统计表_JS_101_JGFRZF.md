# 第一部分：发文原文要求（人行规范层）

> **用于回答"这个表要求报送什么"、"新增业务应该报到哪个表"等问题**

## 1. 官方报送范围

本表统计金融机构（法人层面）的资产负债及风险指标，包括资产总计、负债总计、所有者权益、不良贷款率、拨备覆盖率、流动性比例等关键指标。

## 2. 纳入与排除口径

| 方向 | 内容 |
|------|------|
| ✅ 纳入 | 法人层面汇总的资产负债项目和风险监管指标。 |
| ❌ 排除 | 分支机构单独报送的指标。 |

## 3. 关键业务要素

资产类（各项贷款、投资、现金及存放中央银行款项等）；负债类（各项存款、同业负债等）；所有者权益；风险指标（不良率、拨备率、流动性比例、杠杆率等）。
