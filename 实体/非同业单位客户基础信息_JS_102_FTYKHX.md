# 第一部分：发文原文要求（人行规范层）

> **用于回答"这个表要求报送什么"、"新增业务应该报到哪个表"等问题**

## 1. 官方报送范围

非同业单位客户是指与金融机构开展业务往来的非同业类企业、事业单位、政府机关等客户。本表报送非同业单位客户的基础信息，以客户为单位逐笔报送。

## 2. 纳入与排除口径

| 方向 | 内容 |
|------|------|
| ✅ 纳入 | 与非同业单位客户（企业、事业单位、政府机关等）发生的业务对应的客户信息。 |
| ❌ 排除 | 同业客户、个人客户。 |

## 3. 关键业务要素

客户基本信息（名称、证件类型、证件代码）、客户分类、行业归属、注册地区、注册资本、企业规模等。

## 3. 业务筛选条件

### 3.1 客户范围限制

```sql
-- 限制客户取数范围为贷款客户
INSERT INTO JS_102_FTYKHX_TEMP04
  SELECT T.CUST_ID
    FROM SMTMODS.L_AGRE_LOAN_CONTRACT T
   WHERE DATA_DATE = IS_DATE;  -- 数据日期
```

### 3.2 科目排除

```sql
WHERE T.ITEM_CD NOT LIKE '301%'    -- 排除存款类科目
  AND T.ITEM_CD NOT LIKE '302%'    -- 排除委托贷款科目
  AND T.ITEM_CD NOT LIKE '303%'    -- 排除特定科目
  AND T.ITEM_CD NOT LIKE '304%'    -- 排除特定科目
```

### 3.3 核销与转让排除

```sql
WHERE (A.CANCEL_FLG = 'Y' OR A.LOAN_STOCKEN_DATE IS NOT NULL)  -- 核销（码表A0010：Y=是(已核销)）或已转让（证券化日期非空=已转让）
  AND NOT EXISTS (SELECT 1 FROM SMTMODS.L_ACCT_LOAN B
                   WHERE B.DATA_DATE = IS_DATE  -- 数据日期等于跑批日期，取当前批次数据
                     AND (B.CANCEL_FLG = 'N' AND B.LOAN_STOCKEN_DATE IS NULL)  -- 未核销（码表A0010：N=否(未核销)）且资产未转让（证券化日期为空=未转让）
                     AND A.CUST_ID = B.CUST_ID)  -- 客户名下所有借据均已核销/转让
```
