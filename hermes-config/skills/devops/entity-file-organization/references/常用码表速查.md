# 常用码表速查

> 本项目常见字段 → 码表编码 → 码值含义的快速参考。
> **注意**：以下码表编码从对应监管集市表定义文件中查得，如有疑问以原始定义文件为准。

## 贷款借据信息表（L_ACCT_LOAN）

引用文件：`065_SMTMODS_L_ACCT_LOAN_贷款借据信息表.md`

| 字段 | 码表 | 名称 | 码值含义 |
|------|------|------|---------|
| `ACCT_STS` | A0005 | 贷款账户状态 | 1=正常、2=逾期、3=结清、9=其他 |
| `FUND_USE_LOC_CD` | A0006 | 贷款资金使用位置 | I=境内（中国大陆）、O=境外（含港澳台） |
| `CANCEL_FLG` | A0010 | 是否标志 | Y=是（已核销）、N=否（未核销） |
| `ACCT_TYP` | A0004 | 贷款账户类型 | 01=个人贷款、0102=个人经营性贷款、0301=贴现、90=委托贷款 |
| `LOAN_GRADE_CD` | A0007 | 五级分类 | 1=正常、2=关注、3=次级、4=可疑、5=损失 |

**无码表的字段/条件**（数值或日期判断，不适用码表）：
- `LOAN_ACCT_BAL > 0` — 贷款余额大于0
- `LOAN_STOCKEN_DATE IS NULL` — 资产证券化日期为空（未转让）
- `LOAN_STOCKEN_DATE IS NOT NULL` — 资产已证券化（已转让）
- `DATA_DATE = IS_DATE` — 数据日期等于跑批日期

## 通用字段

| 字段 | 说明 |
|------|------|
| `DATA_DATE = IS_DATE` | 数据日期等于跑批日期，取当前批次数据 |
| `ITEM_CD` | 会计科目号，无统一码表（各业务域有不同科目体系） |
| `DRAWDOWN_DT` | 放款日期，用于发生额时间维度过滤 |
| `EFF_FLAG` = 'Y' | 生效标志（用于中间表/配置表） |

## 注释格式标准

```sql
-- 码表<编码>（<名称>）：<码值>=<含义>（<保留/排除说明>）

-- 正例：
AND A.ACCT_STS <> '3'           -- 码表A0005（贷款账户状态）：3=结清（排除），保留1=正常/2=逾期/9=其他
AND A.CANCEL_FLG = 'N'          -- 码表A0010（是否标志）：N=否（未核销），排除Y=是（已核销）
AND C.CUST_TYP <> '3'           -- 码表C0003（对公客户分类）：3=个体工商户（排除）
AND A.FUND_USE_LOC_CD = 'I'     -- 码表A0006（贷款资金使用位置）：I=境内（中国大陆），排除O=境外（含港澳台）

-- 数值/日期条件（无码表）：
AND A.LOAN_ACCT_BAL > 0         -- 贷款余额大于0
AND A.LOAN_STOCKEN_DATE IS NULL -- 资产证券化日期为空，即资产未转让
AND A.DATA_DATE = IS_DATE       -- 数据日期等于跑批日期

-- 特殊业务条件：
AND T1.GUAR_CONTRACT_STATUS = 'Y'  -- 担保合同状态：Y=有效
```
