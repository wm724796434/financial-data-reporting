# 金融系统文件名缩写翻译字典

适用于金数（PBOC金融基础数据报送）类项目。文件名中的缩写通常遵循：

## 编号体系（业务域）

| 编号 | 业务域 | 说明 |
|------|--------|------|
| 101 | 金融机构信息 | 分支机构、法人信息 |
| 102 | 客户信息 | 个人/单位/同业客户 |
| 201 | 贷款业务 | 对公/个人/贴现/委托/专项/担保 |
| 202 | 存款业务 | 单位/个人/同业存款 |
| 203 | 债券投资 | 债券持有与交易 |
| 205 | 表外业务 | 承兑汇票/保函/票据贴现 |

## 业务类型缩写

### 贷款类（201号段）

| 缩写 | 全称 | 含义 |
|------|------|------|
| CL | 存量 (CunLiang) | 某一时点的数据快照 |
| FS | 发生额 (FaShengE) | 某段时间内的变动数据 |
| DW | 单位 (DanWei) | 对公客户 |
| GR | 个人 (GeRen) | 个人客户 |
| WT | 委托 (WeiTuo) | 委托贷款 |
| TY | 同业 (TongYe) | 金融同业客户 |
| ZX | 专项 (ZhuanXiang) | 专项贷款 |
| DBHT | 担保合同 (DanBao HeTong) | 担保合同信息 |
| DBW | 担保物 (DanBaoWu) | 担保物/抵押品信息 |
| TYJD | 贴现 (TieXian) / 同业借贷 | 票据贴现或同业借贷 |
| ZHJZ | 轧差净额 (ZhaChai JingE) | 置换旧债发生额 |

### 五大篇章核对类（HDA=核对）

| 缩写 | 全称 | 含义 |
|------|------|------|
| PH | 普惠 (PuHui) | 普惠金融贷款 |
| SZ | 数字 (ShuZi) | 数字贷款 |
| YL | 养老 (YangLao) | 养老贷款 |
| KJ | 科技 (KeJi) | 科技贷款 |
| LS | 绿色 (LvSe) | 绿色贷款 |
| HL | 惠农 (HuiNong) | 惠农贷款/互联网贷款 |

### 存款类（202号段）

| 缩写 | 含义 |
|------|------|
| DWCKFS | 单位存款发生额 |
| TYCKFS | 同业存款发生额 |
| FTYDWC | 非同业单位存款 |
| CLGRCK | 存量个人存款 |
| CLTYCK | 存量同业存款 |
| WQYBJY | 未确认一笔交易（代发工资） |
| HDACLHLCK | 惠农存款核对（互联网存款） |

### 表外业务类（205号段）

| YHCDFS | 银行承兑汇票发生额 |
| CLYHCD | 存量银行承兑汇票 |
| PJRZFS | 票据贴现发生额 |
| CLPJRZ | 存量票据融资 |
| ZTXFS | 保贴函发生额 |
| CLZTX | 存量保贴函 |

## 后缀/前缀约定

| 后缀 | 含义 |
|------|------|
| _HIS | 历史镜像版本（取 sysdate-1 历史快照数据） |
| _TMP | 临时表（加工过程中间表） |
| _SQ | 上期数据表（取上期报送数据） |
| _TSCL | 特殊处理（TeShu ChuLi） |
| _ALL | 全量（所有数据） |
| bsp_ | 银行BSP系统前缀 |
| sp_pbocd_ | PBOCD系统日志/工具存储过程 |

## Oracle常用表前缀

| 前缀 | 含义 |
|------|------|
| SMTMODS. | 核心系统（源系统） |
| L_CUST_ | 客户信息表 |
| L_ACCT_ | 账户/借据表 |
| L_PUBL_ | 公共信息表（机构等） |
| PBOCD_ | PBOC报送系统接口表 |
| PBOCD_DATACORE. | PBOC数据核心模式 |
