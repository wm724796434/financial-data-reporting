# 金融基础数据报送系统

金融基础数据（EAST/PBOCD）监管报送系统的存储过程和 SQL 脚本库。

## 目录结构

| 目录 | 说明 |
|------|------|
| `首页/` | 58 个报表实体的完整索引（入口） |
| `实体/` | 每个报表的发文要求 + 代码取数实现说明 |
| `源码/` | 原始存储过程及 SQL 脚本备份 |
| `源码解析/` | 带增强注释的源码副本 |
| `参考资料/发文原文/` | 金融基础数据采集规范 V2.1 原文 |
| `参考资料/监管集市/` | 191 张监管集市表的字段级定义 |
| `参考资料/数据源审计/` | 数据源合规性审计报告 |

## 快速开始

从 `首页/index.md` 开始，按业务域查找目标报表。

## 数据来源

基于人行《金融基础数据采集规范 V2.1》，结合行内监管集市（SMTMODS）实现数据加工。
